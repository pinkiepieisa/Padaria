# Padaria
Repositório para guardar a lição do dia 14 de novembro de 2025 para a matéria de Desenvolvimento Web 3.

Back-end em Node.js + Front-end em React

Como rodar o back:
1 - cd API_node

2 - Instale as seguintes dependências:
    express
    sequelize
    mysql2
    cors
    body-parser
    nodemon

3 - Ajueste o arquivo db.js de acordo com os dados do seu banco mysql

4 - Caso não tenha, faça o database padaria no mysql

5 - Rode a API:
npx nodemon index.js

Reposta:
Servidor rodando na porta: 8081!

6 - Abra um novo terminal

7 - Dê um:
cd padaria-front

8 - Instale as dependências:
    react
    react-dom
    react-router-dom

9 - Rode o front:
npm run dev
